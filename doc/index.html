The italic signature rule

Every page headline should include 2 to 4 words set in Lora italic (<span class="ital">) inside the Poppins headline. This is the typographic signature of the brand and must never be omitted.

Developer Notes
Before you build, read these.

Nine critical points your developer should understand before integrating these templates into the actual CMS or website stack.

🔤
Google Fonts
Templates load Poppins, Inter, and Lora from fonts.googleapis.com. They require an internet connection to render brand fonts. In production, self-host the font files for Core Web Vitals (eliminates render-blocking external request). Use <link rel="preconnect"> already included in each template.
🤖
Aarya AI Chat
The chat widget in chat-widget.js is a static demo with hardcoded responses. In production, replace the getReply() function with an API call to the LLM backend (Claude claude-sonnet-4-6 via Anthropic API or equivalent). Pass the current page URL and context in the system prompt so responses are page-aware.
📋
Lead Forms
All forms fire alert('Lead captured (demo).') on submit. Replace with your actual CRM webhook (fetch('/api/leads', {...})). Tag each form with the source template name and page path so leads can be attributed correctly in the CRM. The 4-field form (Name, Phone, Course, City) is the standard.
🏷
Schema Markup
All schema is in <script type="application/ld+json"> blocks in the <head>. Validate every page with Google's Rich Results Test before launch. The FAQPage schema is the most critical — it enables Google FAQ rich snippets and is what AI engines parse first.
📱
Mobile Breakpoints
The responsive breakpoints are: 880px (nav collapses, grids go single-column, mobile sticky bar appears) and 660px (smaller adjustments, tables reflow). All tested. The mobile sticky action bar adds padding-bottom: 64px to the body automatically via CSS media query.
⚡
Core Web Vitals
Targets are LCP <2.5s, INP <200ms, CLS <0.1. Templates achieve this by design (no layout shifts, no heavy JavaScript). In production: use loading="lazy" on images below the fold, convert images to WebP, inline critical CSS for above-fold content, and serve via CDN. The Lora/Poppins self-hosting will help significantly with LCP.
🔗
Internal Links
All internal links currently point to # placeholders (except inter-template links within the gallery, which use relative paths). Before going live, replace every href="#" with the correct URL. The sitemap and content strategy document (v2.0) provides the full URL structure for every page type.
🗺
301 Redirects
All 21+ old blog-style URLs must have 301 redirects to the new URL structure before launching. See Appendix A of the Sitemap & Content Strategy v2.0 for the complete migration map. Never delete a URL that has indexed search traffic — always redirect it.
📄
llms.txt File
Deploy the llms.txt file at https://distancembacollege.com/llms.txt on day one. The full template is in Appendix D of the Sitemap & Content Strategy v2.0. This file is read by ChatGPT, Claude, Perplexity, and Gemini to understand which pages on the site are most authoritative to cite.
